O que é e como usar as seguintes tags:

Table:

A table tag define uma tabela HTML,consiste em um elemento ou mais elementos . Além de suportar atributos Globais e Eventos em HTML.Exemplo:

Mes gastos:
Fevereiro $1.600

Del:

O Del tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído. Fica por tanto assim:

Minha cor favorita é rosa preto!

Ins:

A INS tag define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido. Exemplo, um texto com uma parte excluída e uma nova parte inserida:

Meu amigo favorito é Vinicius Ruan!

Sub:

 A Sub tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. Exemplo:

Olhha este texto que legal!

Sup:

A Sup tag define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. Exemplo:

Olha este texto que legal

Blocktote:

A blockquote tag especifica uma seção que é citada de outra fonte.Os navegadores geralmente recuam blockquote elementos.

Q:

A q tag define uma cotação curta. Os navegadores normalmente inserem aspas ao redor da cotação.Exemplo:

Como é nossa natureza: Muito linda e esbelta Vamos cuidar dela.

Abbr:

A tag Abbr é usada para uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Address:

A tag Address é usada para as informações de contato do autor,proprietário de um documento ou artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc. O texto no exemplo:

elemento geralmente é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois do
elemento.

Cite:

A cite tag define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.). Nota: O nome de uma pessoa não é o título de uma obra. O texto no cite elemento geralmente é renderizado em itálico .

Bdo:

A tag Bdo significa substituição bidirecional. A tag é usada para substituir a direção do texto atual.

Code:

A tag Code é usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Dica: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS .Exemplo:

O HTML usa tag para ser definido.

Pre:

A tag Pre é usada para define o texto pré-formatado.O texto em um pre elemento é exibido em uma fonte de largura fixa e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML. Exemplo:

            Isto é um exemplo da tag pre.

Kbd:

Esta tag Kbd é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo:

Press Ctrl + C to copy text (Windows).

Precione Cmd + C Copie o texto.
